  • Maple Tree Literary Supplement (MTLS) is a cutting-edge 21st century triannual online journal. MTLS promotes Canadian writing and allied Canadian arts – and occasional scholarship – with emphasis on culturally diverse contribution.

    As technical advisor and web developer for MTLS it was my job to solve some pressing technical problems:

    • The previous twelve journal issues were each developed on a seperate installation of wordpress. This caused maintenance issues and duplicate content across issues.
    • There was no homepage, MTLS.com redirected to the latest issue. Without a prominent homepage, MTLS did not have a good presence in google. Searching for the journal returned older issues.
    • MTLS has a community on facebook but the board members felt that writers and artists would appreciate a private community / social network. So a private social network needed to be created.

    Solution:

    • Extracted the duplicat content (about, contant, archive and submissions pages) and added a home page. Going to mtls.ca now displayed information about the journal and a table of contents from the latest issue. The result was a better presence in Google and a better user experience.
    • Wordpress multisite was used to host multiple issues on one Wordpress instalation. Now with one click it's possible to get a new issue up with the right Wordpress back-end and front-end look. Only content needs to be added, removing the need for a web developer during this task.
    • Buddypress (Wordpress plugin) was used to achieve the community functionality. The community is a social space (like facebook and twitter) except for artists and writers.
    • In addition, I also recreated the original website design but with cleaner markup and modern front-end best practices. The website loads faster, has fewer errors and is responsive.

    Services/Technology: Wordpress, Buddypress, HTML, CSS, SEO, Deployment (shared hosting), and technical advice.
